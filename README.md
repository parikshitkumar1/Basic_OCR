# OCR
## Motivation
to make an OCR using CV libraries
## Requirements
Python 3.8 or above with all [requirements](requirements.txt) dependencies installed. To install run:
```python
$ pip3 install -r requirements.txt
```
## To run
```python
$ python3 OCR.py
```
